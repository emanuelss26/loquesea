# General
## Iniciar
Sobre la carpeta del proyecto (recuerden tener instalado maven)
```
mvn spring-boot:run
```
## Enlaces
Para swagger:
```
http://localhost:9090/swagger-ui.html
````
Para la consola H2 (Es una base de datos volatil)
```
http://localhost:9090/h2-console
```
